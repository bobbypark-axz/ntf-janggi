장기판

9×10 격자, 점(교차점)에 기물 배치. 양 끝 3×3은 궁성, 중앙은 강(楚河漢界). 직선/대각선 격자선은 어두운 갈색.

치수와 구조

  • 치수: 9열 × 10행, 패딩 = cell × 0.6, 기물 크기 = cell × 0.94
  • 배경: radial-gradient(120% 80%, woodLight → woodMid 65% → woodDark)
  • 격자선: stroke woodLine #3A2511, stroke-width 1.1
  • 궁성: 양 끝 3×3, 대각선 X 표시
  • 강: 5번째 줄 위쪽 빈 칸에 한자 "楚河" / "漢界" (Noto Serif KR 700, 격자선 색)
  • 위치 마크: 졸/포 시작점 4개 모서리에 ┐┌┘└ 브래킷
  • 외곽 그림자: 0 14px 32px rgba(0,0,0,0.45) + inset bevel

  • 화면: 세 번째 스크립트 — HomeScreen / SetupScreen / GameScreen / ResultScreen / HelpScreen + MenuSheet
  • 원본 시안 (다크 톤): Janggi Mobile UI.html + *.jsx + assets/*.css

키 인터랙션 명세

  • 기물 선택: 내 진영 기물 탭 → 초록 8각 후광 + 합법 수 표시
  • 이동: 합법 수 칸 탭 → 240ms ease-out으로 위치 이동, 잡힘 기물 페이드아웃
  • 선택 해제: 같은 기물 또는 빈 칸 탭
  • 장군: 보드 외곽 1.2s 적색 점멸 + 중앙 "장군!" 오버레이 1.4초
  • 전환: 화면 진입 fade-in 320ms cubic-bezier(0.22, 1, 0.36, 1)
  • 메뉴 시트: 하단에서 슬라이드업 320ms, backdrop 페이드 200ms

반응형 브레이크포인트

  • 모바일 (≤599px): #root 풀폭, 100dvh 높이, 자체 스크롤
  • 데스크탑 (≥600px): #root 480px 최대, height min(880px, 100vh-48px), 둥근 28px 프레임
